Condição (If - Else)

Exemplo do Algoritmo:


A ← 2
B ← 3
se A > B
    então C ← A + B
fim se

Sintaxe em C:


if (condição V){
    ...
} else {
    ...
}

Exemplo do Algoritmo em C:


#include <stdio.h>
        
int main(){
   int A, B, C;
   A=2;
   B=3;
   if (A > B){
      C = A + B;
   } else {
      C = A - B;
   }
   return 0;
}

Estrutura de Repetição

Enquanto .. faça (While):

Exemplo do Algoritmo:


enquanto <condição> faça
    <sequência de comandos>
fim enquanto

Sintaxe em C:


while (condição) {
    ...
}

Exemplo do Algoritmo em C:


#include <stdio.h>
        
int main(){
    int I;
    I=1;
    while(I<=3){
        printf("I = %d", I);
        I++;
    }
    return 0;
}

Para .. faça (For):

Exemplo do Algoritmo:


para i de a até b faça
    <comando 1>
    <comando 2>
    <comando 3>
    <comando n>
fim para

Sintaxe em C:


for (I=n;I<=n;I++){
    ...
}
//I pode ser qualquer letra, 
pois é um identificador de variável

Exemplo do Algoritmo em C:


#include <stdio.h>
#include <math.h>
    
int main(){
    int I, X;
    I=1;
    X=1;
    for (I=1;I<=5;I++){
        pow(X, 2);
        printf(X);
    }
    return 0;
}

Repita .. até (Do While):

Exemplo do Algoritmo:


repita
    <sequência de comandos>
até <condição>

Sintaxe em C:


do{
...
} while (I <= n);
//I pode ser qualquer letra, 
pois é um identificador de variável

Exemplo do Algoritmo em C:


#include <stdio.h>

int main() {
    int I = 1;
    do {
        printf("I = %d\n", I);
        I = I + 1;
    } while (I <= 3);
    return 0;
}
