Algoritmo

Sintaxe:


   início
     sequência de comandos
   fim

Exemplo:


   início
     caractere nome1, nome2
     escreva(“Digite o primeiro nome: ”)
     leia(nome1)
     escreva(“Digite o segundo nome: ”)
     leia(nome2)
     escreva(nome1, nome2)
   fim

Lê-se:


   A=2
   B=3
   Se A é maior do que B
     Então C = A + B
   Fim Se

Condição

Sintaxe:


   se <condição V>
     então <comandos>
   fim se

Exemplo:


   A ← 2
   B ← 3
   se A > B
     então C ← A + B
   fim se

Lê-se:


   A=2
   B=3
   Se A é maior do que B
     então C = A + B
   fim Se

Alternativa Composta

Sintaxe:


   se <condição V>
     então <comandos>
     se não <outros comandos>
   fim se

Exemplo:


   A ← 2
   B ← 3
   se A > B
     então C ← A + B
     se não C ← A - B
   fim se

Lê-se:


   A=2
   B=3
   Se A é maior do que B
     Então C = A + B
     Se não C = A - B
   Fim Se

Estrutura de Repetição

Enquanto .. faça:

Sintaxe:


   enquanto <condição> faça
     <sequência de comandos>
   fim enquanto

Exemplo:


   I ← 1
   enquanto I <= 3 faça
     escreva(“I = “, I)
     I ← I + 1
   fim enquanto

Lê-se:


   I=1
   Enquanto I é menor ou igual a 3, faça:
     Escreva(“I = “, I)
     I = I + 1
   Fim Enquanto

Para .. faça:

Sintaxe:


   para i de a até b faça
     <comando 1>
     <comando 2>
     <comando 3>
     <comando n>
   fim para

Exemplo:


   X ← 1
   para I de 1 até 5 faça
     X ← X^2
     escreva(X)
   fim para

Lê-se:


   X=1
   Para I de 1 até 5 faça
     X = X^2
     Escreva(X)
   Fim para

Repita .. até:

Sintaxe:


   repita
     <sequência de comandos>
   até <condição>

Exemplo:


   I ← 1
     repita
     escreva("I = ", I)
     I ← I + 1
   até I > 3

Lê-se:


   I=1
   repita
     escreva(“I = “, I)
     I = I + 1
   até I > 3
