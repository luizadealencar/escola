Exemplo de Criação de Tabelas em SQL!

Funcionario
codigo_func nome_func
01 João
02 Maria

Projeto
codigo_prof nome_proj
100 viva
200 natureza

É Alocado
codigo_func codigo_proj
01 100
02 200

Funcionario:


create table Funcionario(
    codigo_func int not null,
    nome_func varchar(30),
    primary key (codigo_func));

Projeto:

                        
create table Projeto(
    codigo_proj int not null,
    nome_proj varchar(30),
    primary key (codigo_proj));

É Alocado:


create table Alocado(
    codigo_func int not null,
    codigo_proj int not null,
    foreign key (codigo_func) references Funcionario(codigo_func),
    foreign key (codigo_proj) references Projeto(codigo_proj));
