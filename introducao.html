Linguagem de C

    #include <stdio.h>
    #include <stdlib.h>
          
    int main(){
        printf("Hello World \n");
        system("pause");
        return 0;
    }

1. Declarações globais: bibliotecas usadas pelo programa:

    #include <stdio.h>
    #include <stdlib.h>
2. Inicio do programa:
    int main(){
3. Escreve uma mensagem na tela de saida:
    printf("Hello World \n");
4. Faz uma pausa antes de terminar o programa:
    system("pause");
5. Retoma o código do erro zero para o Sistema Operacional informando que está tudo ok:
    return 0;
6. Fim do programa:
    }

Indentação

A indentação é o espaçamento (ou tabulação) colocado antes de começar a escrever o código na linha. Ele tem como objetivo indicar a hierarquia dos elementos. No nosso exemplo, os comandos printf, system e return possuem a mesma hierarquia (portanto o mesmo espaçamento) e estão todos contidos dentro do comando main() (daí o porque do espaçamento).

Biblioteca

A biblioteca é o comando #include utilizado para declarar as bibliotecas que serão utilizadas pelo programa. Abaixo temos dois exemplos do uso do comando #include:

  • #include <stdio.h> - adiciona uma biblioteca do sistema: stdio.h (que contém as funções de leitura do teclado e escrita em tela).
  • #include "D:\Programas\soma.h" - adiciona uma biblioteca de nome soma.h, localizada no diretório "D:\Programas\".

Escreva

Quando for texto:


printf("Texto")

Quando for valores:


printf("%tipo_de_saida", expressão1);
printf("%tipo1 %tipo2", expressão1, expressão2);
Tipo de Saída Descrição
%c escrita de um caractere
%d ou %i escrita de números inteiros
%u escrita de números inteiros sem sinal
%f escrita de número reais
%s escrita de vários caracteres
%p escrita de um endereco de memoria
%e ou %E escrita em notação científica

Exemplos:

int x = 10;
float y = 5.0;
printf("%d %f\n", x, y);
                            
int x = 10;
printf("Total = %d\n", x);
printf("%d caixas\n", x);
printf("Total de %d caixas\n", x);

Leia

Alguns "tipos de entrada"
%c Leitura de um caractere
%d ou %i Leitura de números inteiros
(%d para signed, %i for signed or unsigned)
%f Leitura de números reais
%s Leitura de uma string (sequência de caracteres) 

scanf("%tipo_de_entrada", &variável);
scanf("%tipo1 %tipo2", &var1, &var2);

Exemplo:

scanf("%d %d", &x, &z);
